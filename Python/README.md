 ##   Projet de formation Python

Ce dossier contient mes programmes Python couvrant des exercices classiques montrant une partie de ce que je sais faire.

Détail des Programmes.


1. **Operateurs_de_sequence.py** ([disponible ici](Operateurs_de_sequence.py))

Objectif : Manipuler une série de mots ou nombres saisis par l'utilisateur.

Fonctionnalités : Inverse l'ordre des éléments.

Utilise des opérateurs de séquence (split, append, boucles).


2. **Listes_avancees.py** ([disponible ici](Listes_avancees.py))

Objectif : Découper une liste selon des règles spécifiques.

Fonctionnalités : Crée des sous-listes (ex : chaque 3ème élément).

Utilise le slicing Python (liste[start:stop:step]).

3. **FizzBuzz.py** ([disponible ici](FizzBuzz.py))

Objectif : Implémenter le jeu FizzBuzz.

Règles :

Une personne tape un nombre, puis l'ordi :

- Affiche "Fizz" pour les multiples de 3.

- Affiche "Buzz" pour les multiples de 5.

- Affiche "FizzBuzz" pour les multiples de 15.

4. **Gestion_budget.py** ([disponible ici](Gestion_budget.py))

Objectif : Comparer les prix des articles avec un budget alloué.

Entrées :

Une liste de prix.

Une liste de noms d'articles.

Un budget par article.

Sortie : Liste des articles abordables.

5. **Compteur_depenses.py**([disponible ici](Compteur_depenses.py))

Objectif : Suivre vos dépenses quotidiennes de manière interactive.

Fonctionnalités :

Ajouter une dépense.

Voir toutes les dépenses.

Calculer le total et la moyenne.

Effacer les dépenses.

Remarques importantes:

- Tous mes programmes sont autonomes et n'ont pas de dépendances externes.

- Les fichiers ont été créés seul et sans IA. (sinon ça n'aurait aucun intérêt)
